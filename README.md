# DG-STAN: Dynamic Graph Spatio-Temporal Attention Network
Official PyTorch implementation of the paper:

**"DG-STAN: A Dynamic Graph Spatio-Temporal Attention Network for Traffic Flow Prediction"**

---

## 🚀 Overview
DG-STAN introduces a unified dynamic graph fusion mechanism, multi-scale temporal convolution, and a global spatio-temporal attention module for accurate traffic flow prediction.

This repository provides:
- DG-STAN pytorch implementation
- Baseline models (STGCN, ASTGCN, STSGCN, DCRNN, DGCN, LSTM, ARIMA)
- Dataset preprocessing scripts
- Training and evaluation code
- Experimental configurations used in the paper
